Sudoku README
=============

Данная программа решает судоку разной сложности.
Открыв программу вы увидете игровое поле, которое можно редактировать.
Вносите в него известные цифры, после чего нажмите Get Solution, чтобы
программа решила судоку. С помощью кнопки Clear вы можете отчистить
поле. В меню File вы можете открывать поле (Open), сохранять(Save) и
выйти (Quit). В меню Edit есть функция отката от решенного судоку к
исходному и наоборот (Restore).

Запуск:
python sudoku.py

Системные требования:
    1) Python 3.4+
    2) PyQt5
