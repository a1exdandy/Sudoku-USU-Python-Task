Sudoku README
=============

Данное приложение предназначено для решения Судоку разной сложности.

sudoku.py - основной файл проекта, описывающий алгоритм решения судоку и пользовательский интерфейс для работы.
test_sudoku.py - unit-test, генерируемый на основе тестов из директории tests
tests - директория с тестами, взятыми из различных электронных ресурсов
